# SULSELGO Backend

Backend REST API untuk aplikasi wisata Sulawesi Selatan.

## Instalasi

Install dependency:

```bash
npm install
```

## Menjalankan Server

```bash
npm start
```

## Menjalankan Test

```bash
npm test
```

## Menjalankan Coverage

```bash
npm test -- --coverage
```
